# 01-PRD.md - 业务蓝图 (Business Blueprint)

## 1. 产品定位与核心业务闭环

TopMentor 是一款聚焦于中产家庭少儿（6-14 岁）的 1 对 1 音视频双语陪伴与成长导师平台。

* **供给端 (Mentor):** 清华、大学等顶级名校在校生，提供口语对练与榜样陪伴。
* **需求端 (User):** 追求高质量教育资源、希望孩子敢于开口的中产家庭家长。

### 核心业务闭环

1. 【家长】充值课时包 → 浏览学霸列表 → 查看 30 秒音视频自荐 → 预约空闲时间槽
2. 【学霸】短信/公众号提醒 → 浏览器打开 H5 → 释放/管理时间槽 → 准时进入 1v1 互动课室
3. 【课室】双端 1v1 音视频 + H5 同步白板绘本 → 倒计时结束下课
4. 【结算】学霸 24 小时内填写《成长报告》→ 平台解冻课时费 → 学霸提现至微信零钱

## 2. 用户角色与核心功能

### 2.1 家长/学员端 (H5 移动 Web)

> **访问方式：** 手机浏览器或微信内打开链接即可，无需安装、无需小程序审核。  
> 地址示例：`https://m.topmentor.com/`（家长端）、学霸端见 §2.2。

| ID | 功能 | 验收标准 |
| :--- | :--- | :--- |
| F-U-01 | 注册登录 | **手机号 + 短信验证码**登录；可填写孩子姓名、年龄、英语基础（微信内可选网页授权，非必须） |
| F-U-02 | 学霸广场 | 瀑布流；按学校/性别/标签筛选；卡片内播放 30s 自荐视频 |
| F-U-03 | 课时账户 | 微信 H5 支付 / 支付宝 H5 购买课时包；余额实时更新 |
| F-U-04 | 时间预约 | 周历展示 45 分钟槽；选槽扣 1 课时；并发预约仅一人成功 |
| F-U-05 | 1v1 课室 | 倒计时、浏览器 WebRTC 音视频、绘本白板、画笔同步（需 HTTPS） |
| F-U-06 | 课后闭环 | 查看成长报告；生成分享海报（可保存图片或分享链接） |

### 2.2 学霸/导师端 (H5 移动 Web)

> **访问方式：** 独立路径或子域名，如 `https://m.topmentor.com/mentor`。

| ID | 功能 | 验收标准 |
| :--- | :--- | :--- |
| F-M-01 | 入驻审核 | 上传证件、录制/上传 30s 视频；审核通过后可被预约 |
| F-M-02 | 时间矩阵 | 周视图 45 分钟格；可标记可约/关闭 |
| F-M-03 | 课务提醒 | **短信**为主；可选微信公众号模板消息（需服务号） |
| F-M-04 | 报告填报 | 课后 24h 内必填；超时影响结算 |
| F-M-05 | 钱包提现 | 显示余额；企业付款至微信零钱 |

### 2.3 运营管理后台 (PC)

| ID | 功能 | 验收标准 |
| :--- | :--- | :--- |
| F-A-01 | 资质复审 | 查看原始材料；通过/驳回；前端脱敏 |
| F-A-02 | 资产中心 | 上传/上下架 H5 绘本课件 |
| F-A-03 | 财务大盘 | 总充值、总消耗、未消耗负债、提现流水 |

## 3. 风控与业务规则

1. **防跳单：** IM/留言禁止微信号、11 位手机号；敏感词替换为 `***`；录像、报告、白板仅在平台内打开
2. **退款：** 未消耗课时包可后台人工退款；已消耗不退
3. **取消预约：** 开课前 2 小时内取消扣 1 课时违约金
4. **课时单位：** 1 课时 = 45 分钟；不足按心跳实际时长比例结算（断线场景）

## 4. 非功能需求

| 类别 | 要求 |
| :--- | :--- |
| 性能 | 预约接口 P99 < 500ms；心跳接口 P99 < 100ms |

## 6. 客户端形态说明（2026-06 决策）

**放弃微信小程序，改用 H5 移动 Web。**

| 对比项 | 微信小程序 | H5 移动 Web（当前方案） |
| :--- | :--- | :--- |
| 上架/审核 | 需微信审核，周期不确定 | **无需审核**，部署即上线 |
| 打开方式 | 仅微信内 | **任意手机浏览器** + 微信内链接 |
| 登录 | wx.login + 手机号组件 | **短信验证码**（主）；微信内可选 OAuth |
| 支付 | 小程序 JSAPI | 微信 H5 支付 / 支付宝 H5 |
| 音视频 | 小程序 SDK | **Agora Web SDK**（Chrome/Safari/微信内置浏览器） |
| 消息提醒 | 订阅消息 | 短信 + 可选公众号模板消息 |

家长端与学霸端为**同一 H5 工程**，通过路由区分（`/`` 与 `/mentor`），运营后台仍为 PC 独立 SPA。
